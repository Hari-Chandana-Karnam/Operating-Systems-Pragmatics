STEP TO CREATE RUNNING PREPARATION 2:

1. Create sample.c from the lab manual page 8.
2. Create makefile using: 'spede-mkmf -q'
3. Create .dli file using: 'make'
4. Run the flash: 'flash MyOS.dli'
5. When in flash: 
	a. Run: 'd' to download the makefile
	b. Run: 'g' to enter GBD
6. When in GDB:
	a. Execute the program using: 'c'
